# DATABASE-KANTIN-ITS

## STEP BY STEP

1. Membuat Database
```SQL
CREATE DATABASE KANTIN_ITS;
```

2. Membuat Table Customer
```SQL
CREATE TABLE CUSTOMER(
	ID_CUST INT NOT NULL PRIMARY KEY,AUTO_INCREMENT
	NAME VARCHAR(20) NOT NULL,
	TYPE ENUM('MAHASIGMA', 'STAFF', 'UMUM') NOT NULL
);
```

3. Membuat Foreign Key
```SQL
CREATE TABLE TENANT(
    ID_TENANT INT AUTO_INCREMENT NOT NULL PRIMARY KEY,
    ID_CUST INT NOT NULL,
    ID_SELL INT NOT NULL,
    FOREIGN KEY (ID_CUST) REFERENCES CUSTOMER(ID_CUST),
    FOREIGN KEY (ID_SELL) REFERENCES SELLER(ID_SELL)
);
```

4. Alter Table
```SQL
ALTER TABLE TENANT 
ADD COLUMN TYPE ENUM ('FOOD', 'DRINK', 'SNACK') DEFAULT 'FOOD',
ADD COLUMN STOCK INT CHECK (STOCK > 70);
```

5. Insert Table
```SQL
INSERT INTO CUSTOMER VALUES(001, 'CATUR', 'UMUM');
```
